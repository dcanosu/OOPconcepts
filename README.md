# OOP Concepts - MySolution

This repository contains a C# project demonstrating Object-Oriented Programming (OOP) concepts, featuring a custom `Date` class with robust validation and formatting logic.

## üöÄ Features

- **Encapsulation**: Private fields with public properties using expression-bodied members.
- **Validation**: Custom logic to ensure valid dates (month ranges, day counts, and leap year detection).
- **Constructors**: Multiple constructor overloads, including a default constructor.
- **Formatting**: Custom `ToString` override for standardized date output.
